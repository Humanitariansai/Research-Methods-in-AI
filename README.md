# **Course Structure & Pedagogy**

The course follows the standard Humanitarians AI **‚ÄúLearn ‚Üí Build ‚Üí Teach‚Äù** cycle:

1. **Start With an Outline**
   Contributors refine this outline collaboratively across the term.

2. **Hands-On Labs First**
   Before any lectures are recorded, volunteers produce:

   * Jupyter notebooks
   * Data analysis walkthroughs
   * Mini research studies
   * Critical reading exercises
   * Reproducible pipelines

3. **Use the Humanitarians Courses GPT for Content Creation**
   Volunteers generate:

   * Lecture scripts
   * Slides
   * Research templates
   * Dataset guides
   * Coding examples
   * Visual layouts

4. **YouTube Video Lectures**
   After labs are stable:

   * Core concept videos
   * Methods deep dives
   * AI ethics discussions
   * Research walkthroughs
   * Fellow-led tutorials

5. **Continuous Updating**
   New volunteers extend:

   * Open datasets
   * Tutorial notebooks
   * Recorded mini-lectures
   * Example research projects
